定时器间歇函数 目标:能够说出定时器函数在开发中的使用场景 网页中经常会需要一种功能:每隔一段时间需要自动执行一段代码，不需要我们手动去触发 例如: 网页中的倒计时 要实现这种需求，需要定时器函数 定时器函数有两种，今天我先讲间歇函数 目标:能够使用定时器函数重复执行代码 定时器函数可以开启和关闭定时器 1.开启定时器 setInterval(函数,间歇时间) 作用:每隔一段时间调用这个函数 间隔时间单位是毫秒 1.函数名不需要加括号 2.定时器返回的是一个id数字 2.关闭定时器 let 变量名 = setInterval(函数,间隔时间) clearInterval(变量名) 需要重新开始请不要用const赋值
